# tic_tac_toe
 A simple Tic Tac Toe game built with flutter
 
 Tic Tac Toe is free classic puzzle game also known as Noughts and Crosses or sometimes X and O.
 The Tic Tac Toe game is a game for two players, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeded in placing three respective marks in a horizontal, vertical, or diagonal row wins the game.
